# toxic-fashion-brand-website
A premium dark-themed fashion brand website featuring immersive animations, interactive canvas effects, custom UI components, and a futuristic streetwear aesthetic.
TOXIC — Fashion Without Mercy

TOXIC is a modern fashion brand landing page designed with a dark cyber-gothic aesthetic. The project showcases immersive UI/UX design through animated backgrounds, custom cursor interactions, interactive product displays, canvas-based visual effects, and responsive layouts.

Features
Custom animated cursor
Interactive particle background
Animated product showcase
Dynamic statistics counters
3D rotating feature section
Responsive design
Smooth scroll animations
Dark luxury fashion branding
Canvas-based visual effects
Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
Canvas API
Responsive Web Design
Project Goal

To create a visually immersive fashion-brand experience that combines modern web technologies with bold streetwear-inspired design principles.
